# 자리픽 (JariPick) — 단지 주차 구역 신청 서비스

> 주민이 원하는 주차 구역을 **직접 신청**하고, 수익은 **단지 기금**으로 돌아갑니다.

---

## 해결하는 문제

- 🅿️ **밤 11시 퇴근, 자리가 없어 15분 뺑뺑이**
- 😤 **"좋은 자리는 항상 같은 사람이"** — 불만 민원
- 💸 **관리사무소 수동 배정**의 분쟁·부담

---

## 솔루션

**입주민 앱**에서 원하는 구역을 **희망 금액**으로 신청 → 월별 확정 → **관리비에 자동 합산**

**관리자 웹**에서 단지 도면 위에 구역 표시 → 라운드 운영 → 자동 정산

---

## 왜 기존 방식보다 좋은가

| 기존 (수동 배정/추첨) | 자리픽 |
|---|---|
| 매달 관리소 직원 업무 부담 | 자동 운영 |
| "왜 저 사람이?" 민원 빈번 | 투명한 금액 기반 배정 |
| 수익은 단지로 환원 안 됨 | **전액 단지 기금** |
| 종이·엑셀 관리 | 이력·분쟁 대응 DB화 |

---

## 수익 모델

- **단지별 월 이용료 (SaaS)** — 세대 수 기준 정액
- 결제대행 수수료 **없음** (관리비 합산)
- 입주민 부담 증가 **없음** (이용 희망자만 참여)

---

## 파일럿 제안 — 오금현대

- **3개월 무료 파일럿**
- 범위: 관리자 웹 + 입주민 앱 체험, 실제 입찰 라운드 1회
- 우리 부담: 개발·운영·맞춤 세팅 전부
- 관리사무소 부담: 주민 명부 1회 제공 + 공지 협조
- 성과 지표: 참여율, 민원 감소, 단지 기금 확보액
- **중단 조건**: 언제든 통보로 종료, 데이터 전량 반환·삭제
